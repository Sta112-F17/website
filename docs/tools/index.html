R is the statistical programming language we will learn in this class. We will interface with R using RStudio, a fully feautred IDE. Go to https://vm-manage.oit.duke.edu/ to access RStudio and log in with your Net ID.
GitHub is a web-based version control repository and Internet hosting service. We will be using GitHub in this course as a collaboration platform and a course management system (i.e. submit homework in GitHub as opposed to on Sakai). In order to get started, create a GitHub account, but before then, read about picking a good username. All course materials, including private repositories for students, are on the GitHub organization.
Slack is the communication platform we'll use for this course. You should have received an email with an invitation link to join the course Slack. All content related course communication will happen on Slack, in the relevant channels. For personal matters (only) you can email the professor.
